# Recipe Images

Drop recipe photos here. Recommended specs:

- Format: JPG or WebP
- Dimensions: at least 1200 × 800px (landscape)
- File size: under 400KB (compress with squoosh.app)

Reference in a recipe's front matter:
```yaml
image: /assets/images/your-photo.jpg
```

If no image is set, the layout shows a neutral placeholder.
